# Devops
Cet site web est un simple portfolio que j'ai fait avec les outils 
HTML, CSS, et un peu de Javascript
## Aperçu

- **Design épuré** avec typographie soignée
- **Dark mode** intégré
- **100 % responsive**
- **Navigation fluide** (smooth scroll)
- **Formulaire de contact** (simulation)
- **Aucune dépendance externe** (sauf Google Fonts)

## Structure du projet
portfolio/
├── index.html       ← Page principale
├── styles.css       ← Tout le style (CSS)
├── script.js        ← Interactivité (JS)
└── README.md        ← Ce fichier

Technologies utilisées

HTML5
CSS3 (Flexbox, Grid, Variables CSS)
JavaScript (Vanilla)
Google Fonts (Inter)
